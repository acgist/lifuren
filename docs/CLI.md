# CLI API

## 命令

```
./lifuren[.exe] 命令 [参数...]
./lifuren[.exe] audio [shikuang] [pred|train] [model_file|model_path] [audio_file|dataset]
./lifuren[.exe] image [wudaozi]  [pred|train] [model_file|model_path] [image_file|dataset]
./lifuren[.exe] embedding [shikuang] dataset
./lifuren[.exe] [?|help]
```

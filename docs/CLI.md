# CLI API

## 命令

```
./lifuren[.exe] 命令 [参数...]
./lifuren[.exe] audio [audio-shikuang] [pred|train] [model audio_file|dataset model_name]
./lifuren[.exe] video [video-wudaozi ] [pred|train] [model video_file|dataset model_name]
./lifuren[.exe] embedding dataset [audio|video]
./lifuren[.exe] [?|help]
```

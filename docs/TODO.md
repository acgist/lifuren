## 代码

```
# 宏名称
所有宏定义使用`LFR_`开头

# 指针名称
*Ptr = 指针
*SPtr = shared_ptr指针
*UPtr = unique_ptr指针
形参可以不用

# 引用名称
*Ref = 引用
形参可以不用

# 全局变量
全局变量全部大写

# 命名空间
所有类型、全局方法、全局变量使用命名空间隔离

# 静态方法/静态变量
文件内部方法和变量使用`static`修饰
```
